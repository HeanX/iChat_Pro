Phase 1 剩余收尾只处理阻断性 bug、文档对齐和演示稳定性，不再新增大功能。


## 2. Phase 2 目标

Phase 2 的目标是把 iChat Pro 从“端到端加密通信 demo”补齐为“可作为通信软件使用的产品雏形”。

Phase 2 不做：

- [ ] 不做 Bot、LLM Agent、Channel、Agent Gateway 的正式能力。
- [ ] 不做完整移动端 App。
- [ ] 不做 Signal Protocol / Double Ratchet 级别的完整高级加密协议。
- [ ] 不做正式生产级多端同步。
- [ ] 不做语音/视频通话。
